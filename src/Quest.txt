1. Что такое регулярные выражения?

Последовательность символов, которая задает шаблон поиска в тексте.
Могут выполнять различные операции, такие как поиск, замена и сопоставление текста с заданным шаблоном.

2. Для чего нужны классы Matcher, Pattern, PatternSyntaxException?

2.1 Matcher предоставляет методы для выполнения операций поиска

2.2 Pattern предоставляет методы для компиляции регулярного выражения из строки в объект Pattern

2.3 PatternSyntaxException - это исключение, которое бросается при ошибке в синтаксисе регулярного выражения.
генерируется при попытке компиляции недопустимого регулярного выражения

3. Зачем в регулярных выражениях используются спецсимволы? Приведите прикладные примеры.

Используется для задания шаблона,которые описывают текстовые последовательности с определенными
свойствами или структурами.

4.Для чего нужны квантификаторы? Привести прикладной пример?

Для того чтобы сделать более гибкое выражение, используется для определения кол. повторений.

5. Для чего нужны диапазоны? Привести прикладной  пример.

Диапазоны в регулярных выражениях используются для указания диапазона символов,
которые могут соответствовать определенному месту в тексте.
